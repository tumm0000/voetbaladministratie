# Slimme Voetbalclub Administratie

Applicatie voor het beheren van een voetbalclub: leden, contributies, wedstrijdplanning en beschikbaarheid van spelers en velden.

## Wat doet dit project?

- **Ledenbeheer** — Registratie en beheer van leden en hun gegevens
- **Contributie** — Inning van contributies en automatische herinneringen bij achterstallige betalingen
- **Wedstrijdplanning** — Planning van wedstrijden met beschikbaarheid van spelers en velden

De applicatie biedt een eenvoudige interface voor bestuur en trainers.
